# code_snippet_generator

A code snippet generator which allows you to select the code language and a lot of theme choices.
- Carbon is used in the backend and frontend was built using Atri framework.

## Deployed app
https://atri-apps.github.io/code_snippet_generator

## How to use this app?

### Step 1: Setup

```shell
git clone https://github.com/Atri-Apps/code_snippet_generator.git

cd code_snippet_generator

pipenv install
```

### Step 2: Start the editor

```shell
pipenv shell

atri start
```
